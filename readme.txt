== Description ==

This plugin allows you to add performance monitoring to your wordpress page within a couple of minutes. In very few clicks you have an uptime monitoring job up and running on your WordPress page!

Some of the features of instant@larm are:

* Synthetic monitoring
* Server locations worldwide
* Several different graphs and reports
* Detailed measurement data
* 100 % independent monitoring
* Immediate alerts
* Monitor API, SMTP, FTP, CITRIX and even User-flows
* Verify Service Level Agreement
* Unlimited number of users
* API
* Mobile Apps
* TimeZone feature

== Installation ==

1. Use the WordPress plugin directory or upload the files to your server in the /wp-content/plugins folder of WordPress.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Click the "Performance monitoring" link in the admin menu.
1. Fill out the data for your wordpress installation.
1. Create your free monitoring job.
